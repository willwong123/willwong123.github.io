javascript的返回值

javascript的返回值是作为一个函数执行完成之后，返回出来的一个结果，并且只能有一个结果，并且肯定会返回一个结果。如果没有设置返回值的情况下，他也是会返回一个值，不过这个是undefined。

如果需要这个返回值的时候，使用return可以调取到返回值。

function show() {
    return a;
}
alert (show());

可以看到返回了a

除此之外，在return的时候，还能进行数学计算。

function show(a,b) {
    return a+b;
}
alert (show(1,2));

这时候返回的值就是3了。

还有一种情况，return=false是函数的终止符，能阻止一些事件的默认行为。当函数执行到这里的时候，函数将结束。

例如，a标签的默认行为是打开一个链接，如果a的click事件中包含了return=false，那么a的打开链接将会被阻止，就不会打开该链接！

函数获取其他函数的返回值

有时候，一个函数的执行需要要用到另一个函数的返回值，就需要执行的时候进行获取别人的返回值了。

function sum(a,b){
    var c = a+b;
    return c;
}
function clear(value){
    alert(value);
}
clear(sum(1,2));

clear(sum(1,2))就实现了clear函数调用sum函数的返回值了。

arguments对象

arguments对象是这个函数的不定参，就是不需要知道函数的参数，也能进行调用。arguments[0]就是调取函数的第一个参数，同理arguments[1]就是第二个参数。

多个参数求和操作：
function sum() {
    var i=0,result=0;
    for (i;i<arguments.length;i++) {
        result += arguments[i];
    }
    return result;
}
alert (sum(1,2,3,4,5));

可以看到返回的是15，也就是全部参数的和。
